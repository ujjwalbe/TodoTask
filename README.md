# TodoTask

To run this project please follow the instruction


## Setup virtual environment 
### On Ubuntu 20.04

  ```python3 -m venv venv```
### Activate virtualenv 
  ``` source venv/bin/activate```
## Install Dependency
  ``` pip install -r requirements.txt```
## Run
  ```
  python manage.py runserver
  ```
## Run Migrations
  ```python manage.py migrate```
